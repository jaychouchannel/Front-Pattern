# 可视化拼图低代码编辑器 (Puzzle Builder)

面向零基础用户的网页可视化拼图低代码编辑器，纯前端实现，打开 HTML 即可在浏览器使用。

## 特性

- 🎨 **拖拽即设计**：在画布上拖拽矩形即可弹出组件选择器，自动填充模板
- 📦 **5 种基础组件**：文本、图片、视频、按钮、卡片
- 🗂️ **多页面管理**：新建/重命名/删除多个页面，按钮可绑定页面跳转
- 🖱️ **所见即所得**：选中后可拖动改位置、八向手柄调整大小
- ↩️ **撤销 / 重做**：支持 Ctrl+Z / Ctrl+Shift+Z，最多 50 步历史
- 👁️ **预览模式**：隐藏编辑框，按钮跳转可直接体验
- 💾 **本地保存**：项目自动保存到 localStorage，刷新不丢失
- 📤 **导出 HTML**：一键导出独立可部署的 HTML 文件，包含响应式缩放
- 📐 **辅助网格**：画布 20px 网格 + 5px 吸附，便于对齐
- 🆘 **新手引导**：首次进入显示使用说明
- 📱 **响应式预览**：预览模式可切换 PC / 平板 / 手机宽度

## 使用方法

1. **运行**：直接双击 `index.html` 在浏览器打开（推荐 Chrome/Edge 最新版）;或将整个目录部署到任意静态服务器
2. **添加组件**：
   - 方式一：点击左侧"组件模板"中的图标，组件会添加到画布左上角默认位置
   - 方式二：在中间画布按下鼠标拖拽框选一个矩形区域，松开后会弹出组件选择弹窗
3. **选中组件**：点击画布上任意组件，右侧属性面板会显示该组件可编辑的属性
4. **移动 / 调整大小**：选中后拖动组件本身可移动;拖动周围 8 个圆形手柄可调整大小
5. **设置按钮跳转**：选中按钮组件，在右侧"跳转目标"下拉中选择"跳转到页面"，然后选择目标页面（须先在左侧新建）
6. **多页面**：左侧底部"页面管理"区域可：
   - 新建页面（点击 "+ 新建页面" 按钮）
   - 重命名（鼠标悬停页面项，点击 ✎ 图标）
   - 删除（点击 🗑 图标，至少保留一个）
   - 切换页面（点击页面项）
7. **预览**：点击顶部"预览"按钮进入预览模式，可点击按钮体验跳转，可切换设备宽度
8. **保存**：点击"保存"按钮将项目写入浏览器 localStorage（防止刷新丢失）
9. **导出**：点击"导出 HTML"按钮，浏览器会下载一个完整可独立运行的 HTML 文件，可直接部署

## 快捷键

| 快捷键 | 功能 |
|---|---|
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z 或 Ctrl+Y | 重做 |
| Ctrl+S | 保存项目到本地 |
| Delete / Backspace | 删除选中组件 |
| Esc | 取消选中 |

## 组件属性详解

### 文本 (text)
- 内容：支持简单 HTML 标签（h1/h2/p/strong/em 等）
- 背景色 / 文字颜色 / 字体大小 / 对齐方式 / 内边距

### 图片 (image)
- 图片地址 URL（默认从 picsum 取示例图）
- 替代文本 alt
- 圆角
- 适配方式：cover（填充裁剪）/ contain（完整显示）/ fill（拉伸）

### 视频 (video)
- 视频地址：可以直接填 `.mp4` 文件 URL，或 YouTube/B 站的 embed URL
- 占位提示文本

### 按钮 (button)
- 按钮文字 / 背景色 / 文字颜色 / 字体大小 / 圆角
- 跳转目标：
  - 不跳转
  - 跳转到页面（选择同项目内其他页面）
  - 外部链接（任意 URL，新窗口打开）

### 卡片 (card)
- 卡片图片 URL / 标题 / 描述 / 背景色 / 圆角

## 导出 HTML 说明

导出的 HTML 文件包含：
- 所有页面的元素布局
- hash 路由（`#page_1`、`#page_2`...），按钮跳转通过 `location.hash` 实现
- 响应式缩放：屏幕宽度小于 1240px 时自动按比例缩放画布以适配视口
- 内嵌一份精简的 CSS 复刻编辑器中的组件样式

可直接部署到 GitHub Pages、Vercel、Netlify 或任何静态托管。

## 数据存储

- 编辑时使用 localStorage 持久化整个项目结构（key: `puzzle_builder_project`）
- 想清空重来：浏览器控制台执行 `localStorage.removeItem('puzzle_builder_project')` 然后刷新

## 注意事项

- 跨域图片需对方服务支持 CORS 或允许外链
- 视频若用 YouTube embed，请使用 `https://www.youtube.com/embed/VIDEO_ID` 格式
- 移动端编辑体验有限，建议在电脑端使用
- 本工具不依赖任何后端，所有功能均在浏览器本地完成
